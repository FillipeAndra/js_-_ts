
## Frameworks frontend de js

### Angular

- framework desenvolvido pela google e que trabalha em tipe script

- ele divide o código em módulos, componentes e serviços.

### React

- framework desenvolvido pelo grupo meta que funciona a base de blocos de construção

	- à exemplos de botões pré-feitos, certas animações e elementos já se fazem presentes, podendo ser combinadas e personalizadas

### Vue.js

 - apresenta elementos que se assemelham tanto a react quanto angular tendo uma flexibilidade de trabalhar com pequenos elementos
