
todas as pertes aqui presentes fazem parte do body do html

## Header

- é a parte do topo do site, geralmente utilizada para colocar:

	- uma faixa com os principais links para redirecionamento, 
	
	- uma barra de pesquisa, 
	
	- a logo da empresa (geralmente com link para  a página principal) 
	
	- outros botões como login e criar conta.

- header: tag utilizada para criação de uma header dentro do body

	- para criar uma lista dentro da header com link para entrar geralmete utiliza-se a tag "nav" e dentro dela se cia uma "ol" ou "ul" 
## Seção de conteúdo 

- parte do site conde se encontra o conteúdo a ser apresentado na página

- uma página pode ter apenas uma seção de conteúdo com subdivisões

	- vale salientar que visualmente geralmente uma seção de conteúdo deve caber inteiramente em uma tela, resultando em pequenas subseções com conteúdo de forma resumida  

- main: tag utilizada para sinalizar a criação da seção de conteúdo do site

	- dentro dela cada subdivisão do main é utilizado a tag "section" e é recomendado de cada section tenha um título que o acompanhe

## Rodapé

- geralmente associado com o local onde se deixa os contatos da empresa

- footer: tag utilizada para criar um rodapé no html
